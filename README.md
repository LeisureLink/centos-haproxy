# leisurelink/centos-haproxy

A CentOS7 based docker image with a recent haproxy.

## What's in the box

This docker image adds to a basic centos 7, a recent (see versions below) haproxy, built from
code.

## Tags (Asset Versions)

* **1.0.0** (Centos 7, haproxy 1.6.4)
